# Algorithms and Data Structures

## Corner Grocer Application

The Corner Grocer project is a C++ application that reads grocery purchase records from a file, counts how often each item appears, and displays frequency information to the user. The original version demonstrated basic file processing, menu navigation, and data output.

## Enhancement Overview

The enhanced version improves the structure, reliability, and usability of the original application. The updates focused on strengthening the program’s data-processing logic, improving consistency in item names, expanding the frequency display, and organizing the code so that it is easier to test and maintain.

The enhancement includes the following improvements:

* Improved item-name handling to reduce inconsistent results caused by spelling and formatting differences.
* Expanded the frequency output so users can review purchase data more clearly.
* Reorganized the application logic into more focused and reusable methods.
* Improved readability through clearer naming, comments, and code structure.
* Added testing to verify expected behavior and confirm the reliability of the enhanced implementation.
* Strengthened file-processing and error-handling behavior.

## Skills Demonstrated

This enhancement demonstrates skills in C++ programming, file input and output, data structures, frequency analysis, algorithmic problem-solving, testing, debugging, code organization, and maintainable software design.

## Course Outcome Alignment

This artifact demonstrates the ability to design and evaluate computing solutions using algorithmic principles and computer science practices. It also shows the use of established development techniques and tools to improve the reliability, clarity, and value of an existing software solution.
